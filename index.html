Wild Horses

This entire chat widget — launcher, popup, event search, WhatsApp share — is added to this page with a single script tag. Look bottom-right, or:

That's all you embed: 
<script src="eventfinder-widget.js"
        data-apikey="YOUR_KEY"
        data-orgid="45742"
        data-venueid="740573"></script>

Programmatic API: EventFinder.open() · .close() · .toggle() · .search("events this week") — see README.md for all options.
